# JourneyCraft Brand Assets

- `journeycraft-icon-flat.png`: safe cross-platform app icon for Expo, Android adaptive icon foreground, older iOS, Expo Go, and web favicon.
- `journeycraft-icon-liquid-glass.png`: iOS 26+ Liquid Glass reference asset. Managed Expo cannot select this PNG dynamically by OS version or package it as an Apple Icon Composer layered icon without native iOS project support. Use this file when creating the future Xcode/Icon Composer icon set.
- `journeycraft-logo-light.png`: horizontal JourneyCraft / 织旅 logo for splash and light in-app brand surfaces.
- `journeycraft-logo-mono.png`: monochrome horizontal logo for single-color or dark/mono brand contexts.

Brand text is `JourneyCraft` in English and `织旅` in Chinese. Do not use incorrect alternate Chinese spellings.
